# Auto Collision Generator — Documentation

## Overview

One-click collision generation for Godot 4. Built for mobile devs, works on desktop too.

## How to Use

1. Open your 3D scene
2. Enable **Editable Children** (for instanced models)
3. Click **Generate Collision**

## Free Version

- Trimesh collision (high precision)
- Supports GLB/GLTF instanced models
- Safe to run multiple times

## PRO Version

- 3 collision types: Trimesh, Convex, Multi Convex
- Auto Editable Children (no manual toggling)
- Node filter & auto replace
- Real-time notifications
